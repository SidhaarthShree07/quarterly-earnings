Code Sample: CAGR Helper

// Compound Annual Growth Rate (CAGR)
// start: Starting value; end: Ending value; years: number of years
function cagr(start, end, years) {
  return Math.pow(end / start, 1 / years) - 1;
}

console.log((cagr(2.85e9, 3.42e9, 1) * 100).toFixed(2) + "%");

Financial Math

Net Present Value (NPV):

\[ \mathrm{NPV} = \sum_{t=1}^{T} \frac{CF_t}{(1 + r)^t} - C_0 \]

Capital Asset Pricing Model (CAPM):

\[ \mathbb{E}[R_i] = R_f + \beta_i\,(\mathbb{E}[R_m] - R_f) \]

Earnings Per Share (EPS):

\[ \mathrm{EPS} = \frac{\text{Net Income} - \text{Dividends on Preferred}}{\text{Weighted Avg. Shares}} \]
